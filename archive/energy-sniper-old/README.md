# Energy Sniper

Mobile-friendly energy/oil trade prep dashboard for Termux and desktop.

## Files
- `energy_sniper.py` — main dashboard script

## Features
- Grouped energy watchlist
- Driver / structure / momentum / alpha layout
- Pullback and breakout levels
- Stops and targets
- Process reminders
- Weekend risk rules

## Run
```bash
python energy_sniper.py
